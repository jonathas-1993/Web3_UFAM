# Minha Página HTML - Execução de Smart Contract

Este projeto demonstra a execução de um Smart Contract diretamente na interface web.

## 🚀 Execução do Smart Contract

O Smart Contract pode ser executado, também, na seguinte plataforma:

🔗 [WebSim - Hardhat Hello World App](https://websim.ai/@JonathasTN/hardhat-hello-world-app)

## 📜 Descrição

Esta página HTML fornece uma interface para interagir com um Smart Contract. Com a tecnologia Web3, é possível realizar operações diretamente pelo navegador, sem necessidade de um ambiente blockchain local.

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Web3.js)
- Hardhat

## 🎯 Como Usar

1. Acesse a página.
2. Conecte-se à carteira Web3.
3. Execute funções do Smart Contract.
